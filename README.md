
# EventHub Authentication Lab

## Setup Instructions

**1. Clone the repository**

```
git clone <repo-link>
cd eventhub-auth-lab
```

**2. Install dependencies**

```
npm install
```

**3. Configure environment variables**

Copy `.env.example` to `.env` and update the MongoDB connection string.

Run the server

```
node src/server.js
```
